Convert Colors Info

Description:

This tool is for converting any color (currently only in the three or six digit hex format) to the closest color in the Material Design color palette. You can switch the output format to any other color format with the "Formats" dropdown.

Instructions:
  • Enter the not Material Design color in the "Your Color" input field.
  • Press "Convert" or hit "Enter" or "Return".
  • Copy to clipboard by pressing the button with the copy icon or press the eyedropper to navigate to color on the Color Picker's page.
